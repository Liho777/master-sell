# MasterSell — Landing Page MVP

Простая landing page для валидации идеи MasterSell: ИИ-инструмент для создания SEO-текстов и инфографики для карточек товаров на Wildberries, Ozon и Яндекс.Маркет.

## Что внутри

- `index.html` — одностраничный лендинг с:
  - Hero-блоком и призывом к действию
  - Описанием двух ключевых фич (SEO + визуал)
  - Как это работает (3 шага)
  - Тарифами (Старт / Профи / Агентство)
  - Формой для сбора email (waitlist)
  - FAQ
  - Адаптивным дизайном (Tailwind CSS CDN)

## Как посмотреть локально

Просто откройте файл `index.html` в браузере:

```bash
# Windows
start index.html

# Или через VS Code — Live Server
```

## Как собирать email подписчиков

Сейчас форма сохраняет email только в `localStorage` и выводит в консоль. Для реального сбора подключите один из сервисов:

### Вариант 1: Google Forms (самый простой)
1. Создайте Google Form с полем Email.
2. Получите пре-заполненную ссылку (Get pre-filled link).
3. Замените action формы на URL Google Form.

### Вариант 2: Tally.so
1. Создайте бесплатную форму на Tally.
2. Вставьте embed-код вместо текущей формы.
3. Ответы приходят в дашборд Tally.

### Вариант 3: Formspree
```html
<form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
  <input type="email" name="email" required />
  <button type="submit">Отправить</button>
</form>
```

### Вариант 4: EmailOctopus / Mailchimp / Unisender
Для полноценной email-рассылки: добро пожаловать, запуск, новости.

## Что валидируем этим лендингом

1. **Есть ли спрос?** Сколько человек оставляют email за неделю?
2. **Какая фича интереснее?** Можно добавить опрос после подписки или A/B-тесты.
3. **Какой тариф выбирают?** Можно отслеживать клики по кнопкам тарифов.
4. **Откуда приходят люди?** Telegram-каналы селлеров, VK, реклама.

## Метрики для отслеживания

Подключите Яндекс.Метрику или Google Analytics и настройте цели:

| Цель | Событие |
|---|---|
| Подписка | Отправка формы `#waitlist-form` |
| Прокрутка до тарифов | Достижение блока `#pricing` |
| Клик по тарифу | Клик по кнопке `Выбрать` |
| Прокрутка до FAQ | Достижение блока `#faq` |

## Что делать дальше

1. **Разместить лендинг** на GitHub Pages / Netlify / Vercel.
2. **Подключить реальный сбор email** (Tally / Formspree).
3. **Подключить аналитику** (Яндекс.Метрика).
4. **Запустить в 3–5 Telegram-каналах** для селлеров с просьбой оценить идею.
5. **Собрать 50–100 email** — это сигнал, что продукт стоит делать.
6. **Параллельно начать разработку MVP**:
   - Функция SEO-генератора карточек
   - Функция визуализации фото в инфографику
   - Авторизация и тарифы
